
HOW TO INSTALL the edu-sharing module "Moodle 2.0"

--------------------------------------------------------------------------------
Install moodle

Please see your "moodle-2.0" distribution on how to achive this.

--------------------------------------------------------------------------------
Change current directory to "moodle-2.0"-install-dir

# cd /your/moodle/installation/dir

--------------------------------------------------------------------------------
Extract "mod_edusharing" into your 

# tar xvfz /path/to/downloaded/mod_edusharing-1.0.tar.gz

--------------------------------------------------------------------------------
Edit /your/moodle/installation/dir/mod/edusharing/config/cs_conf.php

# vi mod/edusharing/config/cs_conf.php
-> set cookie-params accordingly

--------------------------------------------------------------------------------
Edit /your/moodle/installation/dir/mod/edusharing/config/homeApplication.xml

# vi /your/moodle/installation/dir/mod/edusharing/config/homeApplication.xml

--------------------------------------------------------------------------------
Edit web-service-descriptions to insert correct service-endpoint-url

# vi mod/edusharing/services/authenticationservice.wsdl
-> <wsdlsoap:address location="http://<MOODLE_SERVER_URL>/mod/edusharing/services/ccauthenticationservice.php"/>

# vi mod/edusharing/services/permissionservice.wsdl
-> <wsdlsoap:address location="http://<MOODLE_SERVER_URL>/mod/edusharing/services/ccpermissionservice.php"/>

--------------------------------------------------------------------------------
reload page -> install plugin and blocks

F5
